# Figure panel map

Updated: 2026-06-29

This map aligns the main and extended figures with the four-module Results structure in `Manuscript_NatureSubjournal_revised.md`.

## Main figures

- Figure 1: discovery and core spatial architecture. Core conclusion: CAF-myeloid-high regions behave as reproducible spatial-statistical tissue cores in PDAC. Evidence chain: cohort scale, random-core specificity, threshold sensitivity, representative null intervals, distance decay and H&E-anchored spatial maps. Key support: Extended Data Fig. 1, Extended Data Fig. 7, Supplementary Tables 3, 4, 4B and 5. Boundary: observational spatial organization, not causality, mature TLS formation or clinical prediction.
- Figure 2: metastatic-site remodeling and lymph-node immune decoupling. Core conclusion: primary tumors and liver metastases preserve CAF-core immune/tumor coupling, whereas five lymph-node metastases retain tumor-aggressive coupling with weaker and more heterogeneous immune/IFN coupling. Evidence chain: GSE272362 site composition, random-core validation by site, CAF-core subprogram decomposition, immune-decoupling index and representative site-level maps. Key support: Extended Data Fig. 2, Extended Data Fig. 8 and Supplementary Table 6. Boundary: five-sample hypothesis-generating LN lead, not a clinical subtype or causal mechanism.
- Figure 3: CAF-core ecotypes, interfaces and mechanism-priority axes. Core conclusion: CAF-core architecture resolves recurrent ecotypes and nominates SPP1/TAM-matrix, TGF-beta/EMT and matrix-integrin-associated interface axes for follow-up perturbation. Evidence chain: NMF ecotypes, context/ecotype composition, immune-decoupling by ecotype, candidate-axis enrichment, candidate-axis correlations and representative interface maps. Key support: Extended Data Figs. 3, 4, 6 and 7, Supplementary Table 7. Boundary: pathway-level nomination and prioritization, not causal ligand-receptor signaling.
- Figure 4: independent validation and model boundaries. Core conclusion: independent Visium and Xenium datasets support a related CAF-domain organization across complementary resolutions while defining what the current data cannot claim. Evidence chain: external validation scale, GSE274557 Visium validation, GSE274673 Xenium fixed-anchor summary, anchor deltas and representative cell-level maps. Key support: Extended Data Figs. 1, 3, 5 and 6. Boundary: no LN-specific validation in GSE274557, no direct CAF-to-tumor epithelial proximity proof in GSE274673 and no causal signaling.
